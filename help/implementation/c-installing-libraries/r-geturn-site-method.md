# getUrn Site Method{#geturn-site-method}

Este método devuelve la URL de este sitio.

## Ejemplo de Java {#section_nyl_ycs_rz}

Uso:

```

```

Salida de ejemplo:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
