# Etiquetas de contenido y colección{#collection-and-content-tags}

Agregue etiquetas de palabra clave a colecciones y contenido para categorizar y organizar el contenido para facilitar su recuperación.

La asignación de etiquetas permite filtrar las búsquedas por una etiqueta específica y extraer datos para la API del índice de calor.

Las etiquetas se pueden agregar a través de los paneles **[!UICONTROL Content]** y **[!UICONTROL Collections]** de Studio, o directamente a los metadatos de la colección (mediante el objeto collectionMeta) al inicializar o actualizar la colección con Livefyre.

## Uso de etiquetas de recopilación con la API de índice de calor {#section_uhd_frk_wy}

La API de índice de calor acepta un parámetro de etiqueta , que le permite recuperar la intensidad relativa de cualquier colección con esa etiqueta, en un sitio o en toda la red.

Para obtener más información sobre la implementación, consulte la documentación técnica de la API del índice de calor.
